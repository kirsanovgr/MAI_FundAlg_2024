print - список всех студентов
search <mode> - поиск студента по критерию
   |mode = id; name; surname; group
sort <mode> - сортировка студентов по критерию
   |mode = id; name; surname; group
fsearch - поиск студента по id и вывод данных в файл
better - вывод всех студентов в файл, чей средний балл больше общего
stop - завершить программу
