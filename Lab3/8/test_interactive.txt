% Этот файл демонстрирует использование различных команд с многочленами.
% Каждый пример включает комментарии, которые объясняют операции.

% Многострочный комментарий
[ 
    В этом примере мы будем работать с несколькими многочленами,
    выполнять операции сложения, вычитания, умножения и другие.
    Результаты будут сохраняться в сумматоре, который используется
    как первый параметр для следующих операций.
]

% Сложение двух многочленов
Add(2x^2 - x + 2, -x^2 + 3x - 1);
% Результат: (2x^2 - x + 2) + (-x^2 + 3x - 1)

% Умножение двух многочленов
Mult(x^2 + 3x - 1, 2x + x^3); 
% Результат: (x^2 + 3x - 1) * (2x + x^3)

% Вычитание одного многочлена из другого
Sub(4x^2 + 3x - 5, 2x^2 - x + 1);
% Результат: (4x^2 + 3x - 5) - (2x^2 - x + 1)

% Использование текущего значения сумматора (по умолчанию это 0)
Add(4x - 8); 
% В качестве первого аргумента будет использован 0 (из сумматора),
% Результат: (2x^2 + 4x - 6) + (4x - 8)

% Вычисление значения многочлена в точке x = 1
Eval(1); 
% Необходимо вычислить многочлен из сумматора при x = 1.

% Дифференцирование многочлена
Diff(1); 
% Производная текущего многочлена (из сумматора).

% Композиция текущего многочлена с собой
Cmps(1); 
% Композиция текущего многочлена с самим собой.

% Деление одного многочлена на другой
Div(x^5, x^2 - 1); 
% Деление x^5 на x^2 - 1

% Остаток от деления многочлена на многочлен
Mod(x^4 + 3x^2 - 2, x^2 - 1); 
% Остаток от деления x^4 + 3x^2 - 2 на x^2 - 1
